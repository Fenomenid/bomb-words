# Слова-мины

Онлайн-игра для компании. Игроки общаются голосом во внешнем созвоне, а сайт ведет комнаты, роли, таймеры, мины и очки.

## Как играть

1. Хост создает комнату и отправляет ссылку приглашения.
2. Игроки заходят по ссылке и вводят имя.
3. Хост выбирает настройки: сложность слов, количество мин, таймеры и условие финала.
4. В каждом раунде один игрок объясняет слово, следующий по кругу угадывает, остальные становятся минерами.
5. Минеры заранее придумывают слова-мины. Объясняющий и отгадывающий мины не видят.
6. Во время объяснения минеры нажимают на сработавшие мины.
7. Если слово угадали, объясняющий и отгадывающий получают по 1 очку.
8. Каждая сработавшая мина снимает по 1 очку с объясняющего и отгадывающего.
9. После результата следующий раунд стартует автоматически, пока не выполнено условие финала.

## Возможности

- Комнаты по приглашению.
- Realtime-синхронизация через Socket.IO.
- Персональное состояние для разных ролей.
- Скрытие мин от объясняющего и отгадывающего.
- Редактирование своих мин до начала объяснения.
- Таймеры с паузой для хоста.
- Темная и светлая тема.
- Включение и отключение звука таймера.
- Уровни сложности: легкий, средний, нереальный.
- По 500 однословных русских карточек на каждый уровень.
- Финал по целевым очкам или количеству раундов.
- Итоговый экран с топ-3.

## Стек

- Frontend: React, Vite, TypeScript.
- Backend: Node.js, Express, Socket.IO, TypeScript.
- Хранение: in-memory.

## Render

Проект рассчитан на деплой как один Node Web Service.

```text
Build Command: npm install && npm run build
Start Command: npm start
```

Environment:

```text
NODE_ENV=production
```

Render сам задает `PORT`. В production сервер отдает собранный `client/dist`, а клиент подключается к Socket.IO на том же домене.

## Ограничения

- Состояние хранится в памяти процесса.
- При перезапуске сервера комнаты и очки сбрасываются.
- Если игрок отключился, он удаляется из комнаты.
- Если отключился хост, хостом становится следующий игрок.
- Если отключился объясняющий во время раунда, раунд завершается как пропущенный.
